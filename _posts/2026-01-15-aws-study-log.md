# AWS Solutions Architect Associate 자격증 공부 시작

올해 목표 중 하나인 AWS SAA 자격증 취득을 위해 본격적으로 공부를 시작했습니다.

## 학습 계획

### 1단계: 기초 다지기 (2주)
- AWS 기본 개념 학습
- 주요 서비스 이해 (EC2, S3, RDS, VPC 등)
- AWS 콘솔 실습

### 2단계: 심화 학습 (4주)
- 아키텍처 설계 패턴
- 보안 및 규정 준수
- 비용 최적화
- 고가용성 및 내결함성

### 3단계: 시험 준비 (2주)
- 모의 시험 풀이
- 부족한 부분 보완
- 최종 정리

## 학습 자료

1. **온라인 강의**: Udemy의 Stephane Maarek 강의
2. **공식 문서**: AWS 공식 문서 읽기
3. **실습**: AWS Free Tier 활용
4. **모의고사**: Tutorials Dojo 문제집

## 오늘 배운 내용

### EC2 인스턴스 타입
- 범용 (T, M 시리즈)
- 컴퓨팅 최적화 (C 시리즈)
- 메모리 최적화 (R, X 시리즈)
- 스토리지 최적화 (I, D 시리즈)

각 인스턴스 타입의 특징과 사용 사례를 정리했습니다.

### S3 스토리지 클래스
- Standard: 자주 액세스하는 데이터
- Intelligent-Tiering: 액세스 패턴이 변하는 데이터
- Glacier: 아카이브용 장기 보관
