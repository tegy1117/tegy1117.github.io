## Project Overview

A machine learning project using TensorFlow. Developing an image classification model.

### Goals

- Collect and preprocess image dataset
- Design and train CNN model
- Evaluate and optimize model performance
- Deploy as web API

### Current Progress

Dataset collection and preprocessing completed, and basic CNN model training in progress.
Currently tuning hyperparameters to improve model performance.

### Tech Stack

- Python 3.9
- TensorFlow 2.x
- NumPy, Pandas
- Jupyter Notebook

### Learning Topics

- Deep learning fundamentals
- CNN architecture
- Data augmentation techniques
- Transfer Learning
