## 프로젝트 개요

TensorFlow를 활용한 머신러닝 프로젝트입니다. 이미지 분류 모델을 개발하고 있습니다.

### 목표

- 이미지 데이터셋 수집 및 전처리
- CNN 모델 설계 및 학습
- 모델 성능 평가 및 최적화
- 웹 API로 배포

### 현재 진행상황

데이터셋 수집 및 전처리가 완료되었으며, 기본 CNN 모델 학습을 진행했습니다.
현재는 모델 성능 개선을 위한 하이퍼파라미터 튜닝 단계입니다.

### 기술 스택

- Python 3.9
- TensorFlow 2.x
- NumPy, Pandas
- Jupyter Notebook

### 학습 내용

- 딥러닝 기초 개념
- CNN 아키텍처
- 데이터 증강 기법
- 전이 학습(Transfer Learning)
