## Project Overview

A personal portfolio website development project using GitHub Pages and Jekyll.

### Key Features

- **Dark Mode Theme**: Sleek GitHub-style dark mode design
- **Responsive Design**: Support for mobile, tablet, and desktop devices
- **Multi-language Support**: Available in Korean and English
- **Project Management**: Progress tracking and history recording
- **Blog Functionality**: Development logs and daily records
- **Tag System**: Connecting projects and blog posts
- **Search Feature**: Full content search support

### Tech Stack

- Jekyll 4.4
- SCSS
- Vanilla JavaScript
- GitHub Pages

### Development Process

Designed the project structure and created dynamic content using Jekyll's Liquid template engine. 
Implemented a GitHub grass-style calendar to visualize activity history.
