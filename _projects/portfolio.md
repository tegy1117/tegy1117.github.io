## 프로젝트 개요

GitHub Pages와 Jekyll을 활용한 개인 포트폴리오 웹사이트 개발 프로젝트입니다.

### 주요 기능

- **다크모드 테마**: GitHub 스타일의 세련된 다크모드 디자인
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기 지원
- **다국어 지원**: 한국어/영어 버전 제공
- **프로젝트 관리**: 진행도 추적 및 히스토리 기록
- **블로그 기능**: 개발일지 및 일상 기록
- **태그 시스템**: 프로젝트와 블로그 글 연결
- **검색 기능**: 전체 콘텐츠 검색 지원

### 기술 스택

- Jekyll 4.4
- SCSS
- Vanilla JavaScript
- GitHub Pages

### 개발 과정

프로젝트 구조를 설계하고, Jekyll의 Liquid 템플릿 엔진을 활용하여 동적 콘텐츠를 생성했습니다. 
특히 GitHub의 잔디 스타일 캘린더를 구현하여 활동 내역을 시각화했습니다.

### 향후 계획

- 검색 기능 개선
- 댓글 시스템 추가
- RSS 피드 최적화
- 성능 최적화
